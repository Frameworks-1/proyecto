# proyecto
tienda

Descripcion 

En las papelerías o supermercados, los clientes en ocasiones no saben dónde se encuentra cada producto o no conocen la distribución del lugar. Papelería Naranjo les ayuda a realizar sus compras en línea durante cada regreso a clases, permitiéndoles encontrar fácilmente los productos escolares que necesitan mediante una barra de búsqueda, hasta agotar existencias.

Requerimientos funcionales

Visualizar la lista de productos disponibles de la papelería.
Registrar productos seleccionados en el carrito de compras.
Filtrar productos de la papelería mediante un campo de búsqueda.
Visualizar la información básica de cada producto (nombre, precio y descripción).
Visualizar recomendaciones de productos relacionados con el producto seleccionado

requerimientos no funcionales

1- Adaptabilidad
la papeleria cuenta con un diseño responsive, garantizando una correcta visualización 
y usabilidad en dispositivos móviles, tabletas y computadoras de escritorio mediante el uso de CSS
2-Persistencia
La aplicación deberá conservar los productos agregados al carrito de compras mediante el uso 
deLocalStorage, incluso si el usuario recarga o cierra el navegador
3-Accesibilidad
La interfaz deberá cumplir con criterios básicos de accesibilidad,
utilizando etiquetas HTML semánticas y roles ARIA para permitir su uso con lectores de pantalla.
